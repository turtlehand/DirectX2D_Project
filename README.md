# 리벤쳐 모작

## 소개
DirectX로 만든 리벤쳐 게임 모작 프로젝트

## 개발 환경
비주얼 스튜디오

## 사용 기술
C++

WINAPI

DirectX

## 게임 실행 방법
OutputFile -> bin -> Client.exe 실행

## 인 게임

### 조작 방법
이동                        방향키 

점프                        Z키 

아이템 사용                 X키
