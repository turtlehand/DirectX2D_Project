# 리벤쳐 모작
DirectX로 만든 리벤쳐 게임 모작 프로젝트

## 소개
리벤쳐는 기본적으로 플랫포머 + 젤다의 전설 느낌의 게임입니다. 추가적으로 매우 다양한고 엽기적인 엔딩을 보는 재미가 있는 게임입니다.

## 개발 환경
비주얼 스튜디오

## 사용 기술
C++

WINAPI

DirectX

## 게임 실행 방법
인게임 버전

OutputFile -> bin -> Client.exe 실행

에디터 버전

OutputFile -> bin -> Client_Debug.exe 실행

## 인 게임

### 조작 방법
이동                        방향키 

점프                        Z키 

아이템 사용                 X키
